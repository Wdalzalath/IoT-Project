# Raspberry Pi

**Raspberry Pi** is a small computer that you can use as anything. Raspberry Pi is a low cost, credit-card sized single board computer based on ARM cortex architecture with a power processor and operating speed and wireless capabilities, Raspberry Pi is a good choice for developing IoT projects and applications. This is a simple **_home automation system_** project using Raspberry Pi.


## Requirements
To run this application, you will need:

* [Raspberry Pi](https://www.raspberrypi.org/) Zero or higher
* Internet connect or wi-fi
* Local database or web server database

**Note:** You will also need access to your _Raspberry Pi_ using monitor or any remote desktop software(like- [VNC](https://www.realvnc.com/en/connect/download/viewer/)) for  configure and connect your _Raspberry Pi_ for **_home automation system_** and **_database_**.

## Usage
To run the application, simply run the following command in your terminal:
* Clone this repository to your _Raspberry Pi_ machine and type to run

```css
python iotRpi.py
```

This will launch to connect the **home appliances** with database and read the database for perform the action.
